# 基于LLM的因子自动挖掘框架搭建总结

## 一、框架设计目标与核心思想

### 1. 设计目标
构建一个**7×24小时自动化运行**的因子挖掘系统，具备以下核心能力：
- **相关性控制**：在截面、时序及风险因子上进行综合控制。
- **成熟因子启发**：通过RAG技术融合已有成功经验。
- **自适应反馈机制**：实现因子的迭代优化。
- **模块化与即插即用**：便于功能扩展与维护。

### 2. 核心创新
- 使用**大语言模型（LLM）** 直接生成具有**金融逻辑可解释性**的因子表达式。
- 改进传统遗传规划（GP）方法，解决其“黑箱”与可解释性差的问题。
- 实现从因子生成、筛选、评估到入库的全流程**自动化与持续化**。

---

## 二、总体架构与关键模块

### 1. 双层循环挖掘机制
- **内层循环（并行生成与初筛）**：针对少量候选因子进行并行化挖掘，快速筛选出在训练期内表现优异的因子。
- **外层循环（评估与控制）**：对初选因子进行收益能力评估，并通过改进的MMR机制控制其与现有因子库的相关性，确保风险与收益来源的互补性。

### 2. 改进的因子相关性控制（MMR+）
- **多维度相关性计算**：
    - 截面相关性 (`Rel_cs`)
    - 时序相关性 (`Rel_ts`)
    - **纳入Barra风险因子**进行系统性风险暴露控制。
- **改进的MMR公式**：
    ```
    MMR(f_i) = λ·IC(f_i) - (1-λ)·max_{f_j∈S_s∪S_m} [α·Rel_cs(f_i,f_j) + (1-α)·Rel_ts(f_i,f_j)]
    ```

### 3. RAG启发与反馈迭代机制
- **RAG启发层**：从成熟因子库中检索相似结构的成功因子，为LLM生成提供“灵感”，平衡**创造性**与**实用性**。
- **反思改进层**：从因子优化过程中提取有效的改进思路（`idea`），形成显式反馈，使后续迭代路径更清晰、可控。

### 4. 严格的数据样本分离
- **样本内（训练与挖掘）**：仅使用 **2010年-2019年** 数据。
- **样本外（验证与入库）**：使用 **2020年1月1日-2025年4月30日** 数据进行最终验证。
- **目的**：彻底避免未来信息泄露，保证因子在真实场景下的稳定性和可移植性。

## 三、因子生成与工程化处理系统

### 1. 定制化的Prompt工程
- 为**量价因子**和**基本面因子**分别设计初始化、改进和Idea提取的Prompt模板。
- 在Prompt中内置**量纲一致性校验要求**，确保LLM输出的表达式具备合理的金融逻辑。

### 2. 基本面因子专用算子库
| 算子类别 | 功能说明 | 典型示例 |
| :--- | :--- | :--- |
| **一元算子** | 对单列时序数据进行变换 | `Abs`, `Log`, `Sign`, `Power` |
| **二元算子** | 对两列数据进行交叉运算 | `Add`, `Sub`, `Mul`, `Div` |
| **截面滚动算子** | 计算股票在滚动窗口上的特征 | `Ref`, `rollMean`, `rollStd`, `WMA` |
| **价值因子算子** | 封装财务指标特有的时序计算逻辑 | `yoy_chg`, `yoy_var`, `mom_last_zscore` |

### 3. 自动化表达式修正器
- **工作原理**：基于语法树（AST）解析与类型推断。
- **核心功能**：
    1. 自动校正函数误用和参数错误。
    2. 根据因子类型自动注入时间字段，对齐时序。
    3. 通过模糊匹配技术，容错识别因子名称与运算符。
- **目标**：大幅提升LLM生成表达式的**可执行率**与**准确性**。

### 4. 多槽位并行生成框架
- **架构**：启动多个独立的“因子槽位”，并行执行“生成→优化→评估”完整流程。
- **协同机制**：
    - 共享中央因子库，通过锁机制安全入库。
    - 共享`idea`经验池，实现知识积累与复用。
- **优势**：在保证线程安全的前提下，极大提升了系统的整体挖掘效率。

## 四、实践效果验证

### 1. 量价因子表现（示例）
| 指标 | 因子1 | 因子2 | 因子3 |
| :--- | :--- | :--- | :--- |
| **IC均值** | -0.09 | 0.06 | -0.11 |
| **多头年化超额收益** | 22.60% | 23.91% | 33.60% |
| **多空净值曲线** | 平稳上升 | 平稳上升 | 优异上升 |
| **分组表现** | 明显分层 | 明显分层 | 明显分层 |

### 2. 基本面因子表现（示例）
- **因子1**：多头年化超额收益达 **18.82%**。
- **因子2**：多头年化超额收益为 **4.36%**。
- 多空净值曲线表现良好，分组效应显著。

### 3. 改进机制有效性案例
- **RAG启发改进**：借鉴成熟因子结构（如`Mean(条件A, N) - Mean(条件B, N)`），优化原有因子，**未破坏可解释性**。
- **反思改进**：
    - 某个因子经改进后，**IC均值从-0.63%提升至4.17%**。
    - **多头年化超额收益率从4.24%提升至10.09%**。

### 4. 因子合成效果
- **量价因子合成后**：
    - IC均值：**0.13**
    - 多头年化超额收益率：**17.40%**
- **基本面因子合成后**：
    - IC均值：**0.02**
    - 多头年化超额收益率：**8.96%**

## 五、框架总结与价值

本框架成功构建了一个**高效、智能、可解释**的自动化因子生产线，其核心价值在于：

1.  **解决了传统方法的痛点**：融合了人工因子的可解释性与自动化方法的高效性。
2.  **实现了全方位的质量控制**：通过相关性控制、样本外验证、量纲校验等多重机制，保障因子质量。
3.  **建立了可持续的进化体系**：通过RAG启发和反馈迭代，使系统具备自我优化和知识积累的能力。
4.  **经过了实战验证**：挖掘出的量价与基本面因子均显示出显著且稳定的超额收益。

**展望**：该框架可进一步扩展至另类数据、跨市场因子挖掘，并与强化学习结合实现动态策略配置，具备广阔的演进空间。
