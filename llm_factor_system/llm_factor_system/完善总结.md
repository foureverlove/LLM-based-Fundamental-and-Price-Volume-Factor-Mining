## 使用说明

### 环境配置
1. 安装依赖：
```bash
pip install -r requirements.txt
```

2. 设置环境变量（可选，也可以在config中配置）：
```bash
export CLOSEAI_API_KEY='your-api-key'
export CLOSEAI_BASE_URL='https://api.openai-proxy.org/v1'
```

### 运行方式
1. **批量挖掘模式**：
```python
from run import AlphaMiningSystem
system = AlphaMiningSystem()
system.run_batch_mining(target_count=100)
```

2. **持续挖掘模式**：
```python
from run import AlphaMiningSystem
system = AlphaMiningSystem()
system.run_continuous_mining()  # 7×24小时运行
```

1. **数据路径**：需要根据实际环境修改`config.py`中的数据路径配置
2. **API配置**：确保LLM API配置正确，否则因子生成会失败
3. **内存管理**：长时间运行时注意内存使用，因子池有最大容量限制
4. **线程安全**：所有共享资源都使用了锁机制，确保线程安全
