# 数据格式说明

## 数据路径配置

所有数据路径在 `config.py` 中配置，默认路径为：

```python
"data_paths": {
    "minute": "D:/量化/data/minute_data",
    "daily": "D:/量化/data/daily_data",
    "barra": "D:/量化/data/daily_data"  # Barra因子和日频数据在同一目录
}
```

## 分钟频数据格式

### 路径
`D:/量化/data/minute_data`

### 文件格式
- 每天一个文件，文件名格式：`YYYY-MM-DD.parquet`
- 示例：`2025-12-16.parquet`

### 数据列
```python
columns = ['time', 'open', 'high', 'low', 'close', 'volume', 'amount', 'date', 'stock_code']
```

### 数据格式说明
- `time`: datetime类型，格式如 `2025-07-01 09:30:00`
- `stock_code`: 股票代码
- `open`, `high`, `low`, `close`: 价格数据
- `volume`: 成交量
- `amount`: 成交额
- `date`: 日期

### 使用示例
```python
from data.minute_loader import MinuteDataLoader

loader = MinuteDataLoader()
# 加载单日数据
df = loader.load_day('2025-12-16')
# 加载日期范围数据
df = loader.load_range('2025-01-01', '2025-12-31')
# 加载指定股票数据
df = loader.load_stock('000001.SZ', '2025-01-01', '2025-12-31')
```

## 日频数据格式

### 路径
`D:/量化/data/daily_data`

### 文件格式
- 每个字段一个文件，文件名格式：`{field_name}.parquet`
- 示例：`close.parquet`, `open.parquet`, `volume.parquet`

### 可用字段
- `close.parquet` - 收盘价
- `open.parquet` - 开盘价
- `high.parquet` - 最高价
- `low.parquet` - 最低价
- `volume.parquet` - 成交量
- `amount.parquet` - 成交额
- `vwap.parquet` - 成交量加权平均价
- `turnover.parquet` - 换手率
- `float_market_size.parquet` - 流通市值
- `outstanding_share.parquet` - 流通股本

### 数据格式
- **Index**: 日期（DatetimeIndex）
- **Columns**: 股票代码（stock_code）
- **Values**: 对应字段的数值

### 使用示例
```python
from data.daily_loader import DailyDataLoader

loader = DailyDataLoader()
# 加载单个字段
close = loader.load_field('close')
# 加载所有字段
all_fields = loader.load_all_fields()
# 获取可用字段列表
fields = loader.get_available_fields()
```

## Barra风格因子格式

### 路径
`D:/量化/data/daily_data`（与日频数据同一目录）

### 可用因子文件
- `lnsize.parquet` - 对数市值因子（LNSIZE）
- `liquidity.parquet` - 流动性因子（LIQUIDITY）
- `MidCap.parquet` - 中盘因子（MIDCAP）
- `MOMENTUM.parquet` - 动量因子（MOMENTUM）
- `BarraValue.parquet` - 价值因子（VALUE）
- `Volatility.parquet` - 波动率因子（VOLATILITY）
- `DividendYield.parquet` - 股息率因子（DIVIDEND）

### 数据格式
- **Index**: 日期（DatetimeIndex）
- **Columns**: 股票代码（stock_code）
- **Values**: 因子值

### 使用示例
```python
from data.barra_loader import BarraLoader

loader = BarraLoader()
# 加载单个因子
lnsize = loader.load_factor('LNSIZE')
# 加载所有因子
all_factors = loader.load_all_factors()
# 获取可用因子列表
factors = loader.get_available_factors()
```

## 数据加载器使用说明

### 在代码中使用

所有数据加载器都支持通过构造函数传入自定义路径：

```python
from data.minute_loader import MinuteDataLoader
from data.daily_loader import DailyDataLoader
from data.barra_loader import BarraLoader

# 使用默认路径
minute_loader = MinuteDataLoader()
daily_loader = DailyDataLoader()
barra_loader = BarraLoader()

# 使用自定义路径
minute_loader = MinuteDataLoader("D:/custom/minute_data")
daily_loader = DailyDataLoader("D:/custom/daily_data")
barra_loader = BarraLoader("D:/custom/daily_data")
```

### 在因子表达式中使用

在因子表达式中，字段名使用以下映射：

- `Close` / `close` → 收盘价
- `Open` / `open` → 开盘价
- `High` / `high` → 最高价
- `Low` / `low` → 最低价
- `Volume` / `volume` → 成交量
- `Amount` / `amount` → 成交额
- `Vwap` / `vwap` → 成交量加权平均价

示例因子表达式：
```python
# 20日均线
Mean(Close, 20)

# 价格动量
Close / Ref(Close, 5) - 1

# 量价相关性
Corr(Close, Volume, 10)
```

## 注意事项

1. **数据对齐**：所有数据加载器会自动确保日期索引为DatetimeIndex格式
2. **股票代码对齐**：在`run.py`中加载数据时，会自动对齐所有字段的共同股票代码
3. **缓存机制**：`DailyDataLoader`和`BarraLoader`都有缓存机制，避免重复加载
4. **文件不存在处理**：如果文件不存在，会抛出`FileNotFoundError`异常，需要确保数据文件存在
